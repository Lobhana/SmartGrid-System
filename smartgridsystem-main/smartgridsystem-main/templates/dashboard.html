Dashboard Help
Dashboard Overview

This dashboard provides real-time monitoring of India's power grid. Here's what each section shows:

  • Total Generation: The total power being generated across all sources in megawatts (MW).
  • Total Consumption: The total power being consumed across all regions in megawatts (MW).
  • Renewable Energy: The percentage of power coming from renewable sources (solar, wind, hydro).
  • Reserve Margin: The excess capacity available above current demand, shown as a percentage.
  • Grid Health: Current status of the grid including frequency and congestion levels.
  • Generation Mix: Breakdown of power generation by source type.
  • Load Trend: Recent trend of power consumption over time.
  • Recent Faults: Latest detected issues or anomalies in the grid.
  • Regional Grid Status: Detailed status for each grid region.
Data Refresh

Data is automatically refreshed every 30 seconds. You can manually refresh using the "Refresh Data" button.

Interpreting Values
  • Grid Frequency: Normal operating range is 49.85-50.15 Hz. Values outside this range may indicate imbalance.
  • Reserve Margin: Healthy systems typically maintain 15-20% reserves.
  • Load Percentage: Values approaching 100% indicate regions nearing capacity limits.
